# Panduan Deploy IIoT Backend ke Hostinger VPS

## Persiapan di VPS (jalankan via SSH)

### 1. Install Docker
```bash
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
newgrp docker
```

### 2. Upload semua file ke VPS
```bash
# Dari laptop kolega, upload folder backend ke VPS
scp -r ./IIoT-Backend user@IP_VPS:/home/user/iiot-backend
```

### 3. Masuk ke folder project
```bash
cd /home/user/iiot-backend
```

### 4. Buat file .env dari template
```bash
cp .env.example .env
nano .env   # edit password sesuai kebutuhan
```

### 5. Buat file password MQTT
```bash
# Install mosquitto-clients dulu
sudo apt install mosquitto-clients -y

# Buat file passwd (ganti iiot_mqtt dan MqttSecret456! sesuai .env)
mosquitto_passwd -c mosquitto_passwd iiot_mqtt
# Akan diminta input password, masukkan nilai MQTT_PASSWORD dari .env
```

### 6. Buka port di firewall Hostinger
Di panel Hostinger → VPS → Firewall, buka port:
- 8000 (API FastAPI)
- 1883 (MQTT)
- 9001 (MQTT WebSocket, opsional)

### 7. Jalankan semua container
```bash
docker compose up -d
```

### 8. Cek status
```bash
docker compose ps
docker compose logs backend   # lihat log FastAPI
docker compose logs mqtt_broker
```

## Konfigurasi Hardware/ESP32
Ganti alamat MQTT broker di firmware dari EMQX ke:
- Host: IP_VPS_HOSTINGER
- Port: 1883
- Username: iiot_mqtt (sesuai .env)
- Password: MqttSecret456! (sesuai .env)

## Perintah berguna
```bash
# Restart semua container
docker compose restart

# Stop semua
docker compose down

# Lihat log realtime
docker compose logs -f backend

# Update backend setelah ada perubahan kode
docker compose up -d --build backend
```
